# 🌙 يارا AI — النسخة المطوّرة (v2)

مساعدة ذكاء اصطناعي للدردشة اليومية والعاطفية، بواجهة بسيطة على طراز ChatGPT، عدة ثيمات، وحماية أمنية جدّية. تعمل عبر **10 مفاتيح Groq بالتناوب**.

---

## ما الجديد في هذه النسخة؟

### 🔒 الأمان
- **حماية بكلمة مرور** لكامل الموقع (اختيارية عبر `SITE_PASSWORD`) — بدونها أي شخص عنده الرابط يقدر يستهلك مفاتيح Groq الخاصة بك.
- **تحديد معدل الطلبات (Rate Limiting)**: 15 رسالة/دقيقة و300/يوم لكل جهاز، لمنع إساءة استخدام المفاتيح أو استنزافها.
- **صفحات الواجهة الحساسة (index.html) لم تعد قابلة للوصول مباشرة** حتى لو حاول أحد تجاوز صفحة الدخول عبر رابط مباشر.
- تحديد حجم الطلب الأقصى (64KB) وطول الرسالة الأقصى (4000 حرف) وعدد رسائل السجل المُرسلة، لمنع إغراق السيرفر.
- أخطاء Groq الداخلية لا تُعرض للمستخدمة أبدًا (فقط تُسجَّل في سجلات السيرفر)، لتفادي تسريب أي تفاصيل تقنية.
- رؤوس أمان HTTP قياسية (X-Frame-Options، X-Content-Type-Options، إلخ).
- مقارنة كلمة المرور بطريقة آمنة ضد هجمات التوقيت (`secrets.compare_digest`).

### ✨ الميزات
- **محادثات متعددة** بشريط جانبي (زي ChatGPT): إنشاء، تسمية تلقائية، تبديل، حذف فردي.
- **6 ثيمات** جاهزة: ليلة وردية، أحلام لافندر، غروب دافئ، محيط هادئ، تفتّح فاتح (وضع نهاري)، وكلاسيكي داكن.
- **الإعدادات**: حذف محادثة واحدة، حذف رسائل المحادثة الحالية، **حذف كل المحادثات نهائيًا**، تصدير أي محادثة كملف نصي.
- **نسخ رد يارا** بضغطة زر، و**إعادة توليد الرد** إذا ما عجبك.
- تصميم متجاوب بالكامل مع الجوال (شريط جانبي منزلق).
- المحادثات تُحفظ محليًا في متصفحك (localStorage) — يعني خاصة فيك وما توصل لأي سيرفر.

---

## 1. المتطلبات
- Python 3.9 فأعلى.
- حساب على [console.groq.com](https://console.groq.com/keys) لإنشاء مفاتيح API.

## 2. التثبيت

```bash
cd yara-ai-v2
pip install -r requirements.txt
copy .env.example .env      # على ويندوز
# أو: cp .env.example .env   # على ماك/لينكس
```

افتحي `.env` وعبّي:

1. مفاتيح Groq (`GROQ_API_KEY_1` إلى `GROQ_API_KEY_10`، ما يشترط تعبئتها كلها).
2. **مهم جدًا:** `SITE_PASSWORD` — ضعي كلمة مرور لحماية موقعك قبل ما تشاركي الرابط مع أي أحد.
3. `SECRET_KEY` — ولّدي واحدًا عشوائيًا بهذا الأمر ثم الصقيه في `.env`:
   ```bash
   python -c "import secrets; print(secrets.token_hex(32))"
   ```

## 3. التشغيل

```bash
python app.py
```

افتحي المتصفح على `http://localhost:3000` — إذا حطيتِ `SITE_PASSWORD` بتنودّين لصفحة تسجيل الدخول أولًا.

## 4. النشر للإنتاج

```bash
gunicorn -w 2 -b 0.0.0.0:$PORT app:app
```

- استضيفيه على Render / Railway / Fly.io وأضيفي متغيرات البيئة من لوحة التحكم (وليس برفع `.env`).
- فعّلي `FORCE_HTTPS=true` في `.env` بعد التأكد أن موقعك يعمل فعليًا خلف HTTPS (كل الاستضافات المذكورة توفر هذا تلقائيًا).
- ⚠️ **لا تنشري الموقع للعامة بدون `SITE_PASSWORD`** — بدونها أي شخص عنده الرابط يقدر يستخدم مفاتيحك بلا حدود عملية (فقط حد المعدل العام).

## 5. هيكل الملفات

```
yara-ai-v2/
├── app.py              # السيرفر: تدوير المفاتيح، تسجيل الدخول، حد المعدل
├── requirements.txt
├── .env.example
├── views/
│   ├── index.html       # الواجهة الرئيسية (محمية بتسجيل الدخول)
│   └── login.html       # صفحة تسجيل الدخول
└── public/              # ملفات ثابتة تُقدَّم عبر /static
    ├── style.css        # كل الثيمات والتصميم
    └── script.js        # منطق الواجهة: المحادثات، الثيمات، الإعدادات
```

## 6. تخصيص شخصية يارا

عدّلي متغيّر `SYSTEM_PROMPT` داخل `app.py` لتغيير أسلوب الردود أو الحدود السلوكية.

## 7. إضافة ثيم جديد

في `public/style.css` أضيفي كتلة جديدة على نمط `body[data-theme="اسمك"]{ ... }` بنفس المتغيرات الموجودة، ثم أضيفي الثيم لقائمة `THEMES` في أول `public/script.js`.
